# 📦 STOCK PRO - Enterprise Inventory System

**STOCK PRO** is a professional-grade Inventory and Sales Management System developed with **Java Swing** and **MySQL**. It features a modern Dark UI, real-time analytics, and role-based security.

## ✨ Key Features
- 🔒 **Secure Authentication:** Passwords protected with SHA-256 Hashing.
- 👥 **Role-Based Access Control:** Separate professional dashboards for Admins and Staff.
- 📈 **Visual Analytics:** Interactive stock level charts using JFreeChart.
- 🛒 **Sales Engine:** Real-time stock deduction and low-stock alert notifications.
- 📄 **Reporting:** Export inventory data to **CSV (Excel)** and print-to-PDF functionality.
- 💾 **Database Backup:** Built-in tool to export database backups directly from the UI.

## 🛠️ Technology Stack
- **Language:** Java (JDK 17)
- **GUI:** Java Swing (Flat Dark Theme)
- **Database:** MySQL (XAMPP)
- **Library:** JFreeChart, MySQL Connector

## 🚀 How to Run
1. Create a database named `inventory_db` in XAMPP.
2. Import the `database.sql` file provided in this repository.
3. Open the project in NetBeans IDE.
4. Add the required JAR files (MySQL Connector & JFreeChart) to the project libraries.
5. Run `InventorySystem.java`.
